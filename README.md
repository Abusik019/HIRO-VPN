# HIRO-VPN

## Запуск проекта

1. Установите зависимости:

```bash
npm install
```

2. Запустите проект в режиме разработки:

```bash
npm run dev
```

3. Откройте локальный адрес, который выведет Vite в терминале.

## Полезные команды

1. Сборка проекта:

```bash
npm run build
```

2. Локальный просмотр production-сборки:

```bash
npm run preview
```

3. Проверка линтером:

```bash
npm run lint
```
